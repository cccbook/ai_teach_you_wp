# A2. 常用 Prompt 模板大全

## 第一部分：網站建設類

### 網站整體規劃

```
【網站類型判斷 Prompt】
角色：你是一個網站策略顧問
任務：根據以下描述，幫我選擇最適合的網站類型和平台
背景：
- 主要目的：[展示/銷售/分享/品牌]
- 目標受眾：[描述]
- 預算：[預算範圍]
- 技術能力：[完全新手/會一點/有經驗]
需求：
1. 推薦最適合的網站類型
2. 推薦 2-3 個適合的平台
3. 說明每個平台的優缺點
4. 估計所需時間和費用
```

```
【網站架構規劃 Prompt】
角色：你是一個 UX 架構師
任務：幫我規劃網站的頁面結構和導航
背景：
- 網站類型：[類型]
- 主要功能：[功能列表]
- 目標受眾：[描述]
需求：
1. 列出所有必要的頁面
2. 設計導航結構
3. 說明每個頁面的主要目的
4. 標示頁面之間的連結關係
```

### 網站文案

```
【Landing Page Hero 文案 Prompt】
角色：你是一個高轉換率的 Landing Page 文案專家
任務：幫我撰寫 Hero 區域的文案
背景：
- 產品/服務：[名稱]
- 主要賣點：[核心價值]
- 目標客群：[描述]
- 行動目標：[CTA]
需求：
1. 主標題（8-12字，痛點或利益切入）
2. 副標題（20字以內，詳細說明）
3. CTA 按鈕文字（5-8字）
格式：用 Markdown 表格呈現
```

```
【品牌故事 Prompt】
角色：你是一個品牌策略專家和文案高手
任務：撰寫一個真實感人的品牌故事
背景：
- 品牌名：[名稱]
- 創辦人：[名字]
- 起源：[為什麼開始]
- 理念：[核心價值]
- 未來願景：[目標]
需求：
1. 結構：起源 → 挑戰 → 轉機 → 現在 → 願景
2. 長度：600-800字
3. 語氣：真誠、不浮誇
4. 加入 1-2 個小故事
```

```
【產品描述 Prompt】
角色：你是一個專業的電商文案專家
任務：撰寫產品頁面的完整文案
背景：
- 產品名：[名稱]
- 類型：[類型]
- 主要功能：[功能]
- 目標客群：[描述]
- 價格：[價格]
需求：
1. 產品標題（包含關鍵字、吸睛）
2. 副標題（一句話賣點）
3. 產品描述（100-150字）
4. 特色Bullet Points（5-7個）
5. FAQ（3-4個常見問題）
```

```
【關於頁面 Prompt】

任務：撰寫「關於」頁面的完整內容
背景：
- 你是誰：[背景]
- 做什麼：[服務/產品]
- 核心理念：[價值觀]
- 經歷亮點：[經驗]
- 為什麼選你：[差異化]

1. 開頭鉤子（引人繼續讀）
2. 背景故事（建立信任）
3. 理念說明（情感連結）
4. CTA（下一步行動）
長度：400-600字
```

## 第二部分：社群媒體類

### Instagram 貼文

```
【Instagram 單篇 Prompt】
角色：你是一個 Instagram 內容策略師
任務：幫我撰寫一篇 Instagram 貼文
背景：
- 帳號主題：[主題]
- 目標：[曝光/互動/轉換]
- 本次內容：[內容主題]
需求：
1. 主文案（100-200字）
2. Hashtag（8-12個相關標籤）
3. 開頭鉤子（吸引往下讀）
4. CTA（留言/分享/點擊連結）
格式：分段呈現，方便複製
```

```
【一週內容企劃 Prompt】
角色：你是一個社群媒體內容策劃
任務：幫我規劃一週的社群內容
背景：
- 帳號：[帳號名/主題]
- 目標受眾：[描述]
- 本週主題：[核心主題]
- 發文頻率：[每天/一週幾次]
需求：
1. 每天一個內容主題
2. 每篇包含：格式、標題方向、CTA
3. 風格多樣化
4. 配合時事/節日（如果有）
格式：用表格呈現
```

## 使用提示

### 如何使用這些模板

1. **複製框架**：把 Prompt 模板複製到文件中
2. **填入背景**：根據你的實際情況填入 `[...]` 中的內容
3. **調整需求**：根據具體需求增刪項目
4. **測試迭代**：根據 AI 回覆調整 Prompt

### 提升效果的方法

- 提供更多具體細節
- 說明不喜歡什麼也很重要
- 要求多個版本可以比較
- 讓 AI 說明思考過程
- 根據回覆逐步深化
