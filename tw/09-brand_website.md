# 9. 建立品牌形象網站

## 品牌形象網站的核心目的

品牌形象網站不同於其他類型網站，它的首要任務是：

> **讓訪客在 5 秒內相信你是值得信賴的**

### 品牌網站必備元素

| 元素 | 目的 |
|------|------|
| 品牌故事 | 建立情感連結 |
| 專業能力展示 | 證明你的實力 |
| 客户見證/案例 | 增加可信度 |
| 明確的聯絡方式 | 降低行動門檻 |

## 使用 Durable 建立品牌網站

### 步驟 1：選擇商業類型

在 Durable 的 AI Builder 中選擇或描述你的業務類型：

```
商業選項：
- 餐廳/咖啡廳
- 自由工作者
- 顧問服務
- 健身教練
- 律師/會計師
- 零售店
- 其他服務業

敘述方式（推薦）：
「我要建立一個 [你的職業/行業] 的品牌形象網站，
主要是用來 [主要目的：展示作品/吸引客戶/建立信任]」
```

### 步驟 2：提供品牌資訊

AI 會引導你提供：

1. **品牌名稱**
2. **標語/Slogan**
3. **提供的服務**
4. **目標客戶**
5. **希望呈現的風格**

### 步驟 3：AI 生成網站

提交後，AI 會生成包含以下頁面的網站：
- 首頁
- 關於我們/關於我
- 服務內容
- 聯絡我們

### 關於頁面設計要點

「關於」頁面是建立信任的關鍵：

1. **開頭**：你是誰，做什麼的
2. **故事**：為什麼你做這件事
3. **經歷**：專業背景、證照、經驗
4. **理念**：你的價值觀和工作方式
5. **照片**：本人照片、工作場景

### 服務頁面設計要點

服務頁面要清楚說明：

1. **服務名稱**
2. **服務說明**（2-3 句）
3. **服務流程**（如何開始）
4. **價格範圍**（可選）
5. ** CTA**：預約/詢問

## AI 幫你寫品牌內容

### 品牌故事生成 Prompt

```
角色：你是一個品牌策略專家和文案高手

任務：幫我撰寫一個真實、有溫度的品牌故事

背景資訊：
- 品牌名稱：[名稱]
- 創辦人：[名字]
- 行業：[行業]
- 主要服務：[服務內容]
- 成立原因：[為什麼開始這個事業]
- 核心理念：[你的價值觀]

需求：
1. 結構：起源 → 挑戰 → 轉機 → 現在 → 未來願景
2. 語氣：真誠、不浮誇、讓人想繼續讀下去
3. 長度：600-800 字
4. 加入 1-2 個小故事或細節
5. 最後要能呼應品牌的核心價值
```

### 服務描述生成 Prompt

```
角色：你是一個專業的行銷文案專家

任務：幫我撰寫服務項目的描述文案

服務資訊：
- 服務名稱：[名稱]
- 目標客戶：[誰需要這個服務]
- 服務內容：[詳細說明]
- 預期效果：[客戶能得到什麼]

需求：
1. 每個服務描述 100-150 字
2. 格式：「服務名稱」+ 3 句描述 + 行動呼籲
3. 語氣專業但不死板
4. 強調對客戶的「好處」，不只是「功能」
5. 用動詞開頭，增加行動感
```

### 客戶見證生成 Prompt

```
角色：你是一個理解客戶心理的文案專家

任務：幫我撰寫幾個客戶見證範例

背景：
- 產業：[你的行業]
- 客戶類型：[典型客戶是誰]
- 常見痛點：[客戶通常遇到什麼問題]
- 服務效果：[你的服務如何解決問題]

需求：
1. 撰寫 3-5 個見證
2. 每個見證包含：一句話總結 + 詳細心得
3. 格式：「引言」—— 姓名、公司、職位
4. 見證要具體、有數據更好
5. 語氣自然，不像廣告
```

## 品牌網站的視覺要素

### 配色選擇

品牌顏色要考慮：

1. **行業特性**：科技用藍、餐飲用暖色、自然用綠
2. **目標客群**：年輕人用亮色、商務用深色
3. **情感連結**：想要傳達什麼感覺？

**常見品牌配色方案**：

| 風格 | 主色 | 輔色 |
|------|------|------|
| 專業商務 | 深藍 #1a365d | 金色 #d69e2e |
| 創意科技 | 紫色 #6b46c1 | 藍色 #3182ce |
| 自然健康 | 綠色 #38a169 | 米色 #faf5eb |
| 時尚精品 | 黑色 #1a1a1a | 金色 #ffd700 |
| 温暖服務 | 橙色 #dd6b20 | 白色 #ffffff |

### 圖片選擇原則

品牌網站的圖片要注意：

1. **一致性**：所有圖片色調、風格要統一
2. **真實性**：最好有真實的工作照片
3. **專業性**：避免用低解析度或業餘感照片
4. **情感連結**：圖片要能引起目標客戶共鳴

### 用 AI 生成品牌圖片

```
用 Midjourney 生成：

「[描述你需要的場景]，照片風格，
專業商業攝影，柔和自然光，
[具體色調], [想要的感覺]」

範例：
「Small business owner meeting with clients in modern office,
photography style, natural lighting,
clean aesthetic, warm and professional atmosphere」
```

## 品牌形象網站範例分析

### 案例 1：健身教練品牌

**目標**：吸引想要健身的上班族

**網站要素**：
- 首頁：教練本人照片 + 成果對比
- 服務：個人訓練、線上課程、營養諮詢
- 特色：時間彈性、專為上班族設計
- 見證：上班族学员的改變故事
- CTA：免費諮詢

### 案例 2：室內設計師品牌

**目標**：吸引想要裝修的屋主

**網站要素**：
- 首頁：作品大圖展示
- 作品集：各類型案件詳細介紹
- 流程：從諮詢到完工的完整流程
- 關於：設計理念、經歷、得獎
- CTA：預約丈量

### 案例 3：律師事務所品牌

**目標**：建立專業可信形象

**網站要素**：
- 首頁：沉穩專業的設計
- 服務項目：民事、刑事、家庭、商業等
- 團隊：每位律師的學經歷
- 新聞：法律知識分享
- CTA：預約免費諮詢

## 讓品牌網站更有說服力

### 增加信任的元素

1. **專業證照/資格**：律師執照、專業認證
2. **媒體採訪**：上過的新聞、專訪
3. **得獎經歷**：業界獎項、比賽成績
4. **合作夥伴**：知名客戶、廠商 Logo
5. **數據說話**：服務人數、成功率

### 常見錯誤

1. **貪多求全**：把所有服務都放上去，反而模糊焦點
2. **太過自我**：一直說自己多厲害，沒有說能幫客戶什麼
3. **太官方**：語氣像政府公文，不親近
4. **圖片隨意**：用了一些不專業的示意圖
5. **缺乏行動**：看完不知道下一步要幹嘛
