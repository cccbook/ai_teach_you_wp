# 4. 註冊與熟悉 AI 設計平台

## 選擇你的起點

在開始之前，你需要選擇一個主要的 AI 網站建構平台。本書推薦從 **Durable** 或 **Wix ADI** 開始，因為它們最符合「AI 生成」的核心概念。

讓我們以 **Durable** 為例，帶你完成註冊和基本設定。

## 註冊 Durable

### 步驟 1：前往官方網站

在瀏覽器中輸入：`https://durable.ai`

### 步驟 2：點擊開始使用

在首頁點擊「Get Started」或「Start Building」按鈕。

### 步驟 3：選擇註冊方式

你可以選擇：
- **Google 帳號登入**：最快速方便
- **Email 註冊**：需要設定密碼

建議使用 Google 帳號登入，省去記密碼的麻煩。

### 步驟 4：填寫基本資訊

註冊後需要填寫：
- **你的名字**：用於稱呼
- **商業類型**：餐廳、自由工作者、電商等（可選「Other」）
- **所在地區**：用於在地化設定

### 步驟 5：完成驗證

有些工具會要求 email 驗證，檢查你的郵箱點擊驗證連結即可。

## Durable 介面導覽

登入後，你會看到 Dashboard（控制面板）。讓我們認識主要區域：

### 左側選單

```
┌─────────────────────────────────┐
│  🏠 Dashboard                   │
│  📄 Sites                       │
│  💰 Billing                     │
│  ⚙️ Settings                    │
│  📖 Help                        │
│                                 │
│  ─────── Analytics ───────     │
│  📊 Traffic                     │
│  📈 Leads                       │
│                                 │
│  ─────── Tools ───────         │
│  🔧 AI Builder                  │
│  📝 AI Writer                   │
│  🖼️ Image Generator             │
└─────────────────────────────────┘
```

### 主要功能區

1. **Dashboard**：總覽你的網站狀態、流量、潛在客戶
2. **Sites**：管理你所有的網站專案
3. **AI Builder**：AI 網站生成工具
4. **AI Writer**：AI 文案生成工具
5. **Image Generator**：AI 圖片生成工具

## 第一次建立網站

### 點擊「AI Builder」

在左側選單點擊「AI Builder」，這是生成網站的核心功能。

### 輸入你的網站描述

Durable 會問你：「What would you like to build?」

這是一個關鍵步驟。你可以輸入：

**簡單範例**：
```
個人攝影師作品集網站
```

**詳細範例**：
```
我要一個簡約風格的個人攝影師作品集網站，
以黑白為主色調，
包含首頁、作品集頁面、關於我頁面、聯絡方式頁面
```

**英文範例**（Durable 主要為英文介面）：
```
A minimalist photography portfolio website with black and white theme,
including home page, portfolio gallery, about me, and contact page
```

### 等待 AI 生成

點擊生成後，AI 會開始工作。這個過程通常需要：
- **快速模式**：約 30 秒到 1 分鐘
- **完整模式**：約 2-3 分鐘

### 查看生成結果

生成完成後，你會看到一個預覽畫面。這就是 AI 為你自動建立的網站！

## Wix ADI 註冊流程

如果你選擇 Wix ADI，流程也很類似：

### 步驟 1：前往 Wix ADI

網址：`https://www.wix.com/adi`

### 步驟 2：點擊「Get Started」

### 步驟 3：回答 AI 的問題

Wix ADI 會問你一系列問題來了解需求：

1. **你要建立什麼類型的網站？**
   - 個人網站
   - 商業網站
   - 線上商店
   - 其他

2. **網站的主要目的是什麼？**
   - 展示作品
   - 販售商品
   - 提供服務
   - 分享內容

3. **你有品牌名稱嗎？**

4. **你想要哪些功能？**
   - 部落格
   - 線上預約
   - 會員系統
   - 聯絡表單

### 步驟 4：AI 生成並調整

回答完問題後，Wix ADI 會生成一個網站，你可以在此基礎上進一步調整。

## 認識網站的基本結構

在繼續之前，讓我們了解一下網站的基本組成部分：

### 常見頁面

| 頁面名稱 | 目的 |
|---------|------|
| 首頁 (Home) | 網站的入口，介紹整體內容 |
| 關於我 (About) | 介紹個人或公司背景 |
| 作品集/服務 (Portfolio/Services) | 展示你的工作或服務內容 |
| 部落格 (Blog) | 分享文章或新聞 |
| 聯絡我 (Contact) | 提供聯絡方式或表單 |
| 產品/商店 (Shop/Products) | 展示商品（電商用） |

### 常見區塊

每個頁面由不同的「區塊」(Block/Section) 組成：

```
┌─────────────────────────────────────┐
│         導航列 (Navigation)          │
├─────────────────────────────────────┤
│                                     │
│         主視覺區 (Hero)              │
│         大圖 + 標題文字              │
│                                     │
├─────────────────────────────────────┤
│                                     │
│         特色區 (Features)            │
│         3-4 個服務或特色卡片          │
│                                     │
├─────────────────────────────────────┤
│                                     │
│         內容區 (Content)             │
│         圖文並茂的詳細說明            │
│                                     │
├─────────────────────────────────────┤
│         頁尾 (Footer)               │
│         聯絡資訊、版權聲明            │
└─────────────────────────────────────┘
```

## 熟悉編輯器介面

大多數 AI 建構平台都有類似的編輯器功能：

### 拖放功能 (Drag & Drop)

找到想移動的元素，按住滑鼠拖到新位置。這是現代網站編輯器的基本功能。

### 點擊編輯

大多數文字可以直接點擊後修改。雙擊文字區塊即可進入編輯模式。

### 右鍵選單

在元素上按右鍵，會出現選項：
- **複製**
- **刪除**
- **編輯設定**
- **移動到其他區塊**

### 設定面板

通常點擊元素後，右側或下方會出現設定面板，可以調整：
- 顏色
- 字體
- 大小
- 間距
- 動畫效果
