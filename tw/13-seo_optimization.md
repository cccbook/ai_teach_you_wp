# 13. AI 生成網站的 SEO 優化

## 什麼是 SEO

SEO（Search Engine Optimization，搜尋引擎優化）是指：

> **讓你的網站在 Google 等搜尋引擎的搜尋結果中排名更前面的技術**

排名越前面，越多人看到你的網站。

### SEO 為什麼重要

| 排名位置 | 點擊率 |
|---------|--------|
| 第 1 名 | ~28% |
| 第 2-3 名 | ~15% |
| 第 4-10 名 | ~5% |
| 第 11 名後 | <1% |

大部分人只會看第一頁的搜尋結果。

## SEO 的基本組成

SEO 主要分為三大部分：

```
┌─────────────────────────────────────────┐
│              技術 SEO                    │
│   網站速度、行動裝置相容、SSL 安全性      │
├─────────────────────────────────────────┤
│              內容 SEO                     │
│       關鍵字、標題、內文、圖片優化        │
├─────────────────────────────────────────┤
│              權威 SEO                     │
│         反向連結、社群分享、品牌信任度     │
└─────────────────────────────────────────┘
```

## AI 輔助關鍵字研究

### 找出目標關鍵字

問 ChatGPT：

```
角色：你是一個 SEO 關鍵字策略專家

任務：幫我找出適合我網站的關鍵字

網站類型：[個人部落格/品牌網站/電商網站等]
主題領域：[你的領域]
目標受眾：[誰會搜尋這個]
主要服務：[提供的內容]

需求：
1. 列出 10 個主要關鍵字（每個 2-4 字）
2. 列出 10 個長尾關鍵字（每個 5-8 字）
3. 說明每個關鍵字的搜尋意圖（資訊/交易/導航）
4. 評估每個關鍵字的競爭程度（高/中/低）
```

### 分析競爭對手關鍵字

```
任務：幫我分析競爭對手的網站可能針對哪些關鍵字

競爭對手網站：[URL]
我的網站主題：[主題]

需求：
1. 猜測他們的主要關鍵字
2. 找出我可以切入的差異化關鍵字
3. 建議我可以超越他們的關鍵字策略
```

## AI 幫你優化標題標籤

### 標題優化原則

| 要素 | 說明 |
|------|------|
| 長度 | 50-60 字以內 |
| 包含關鍵字 | 越前面越好 |
| 吸引點擊 | 讓人想點 |
| 品牌名 | 可放在最後 |

### AI 協助撰寫

```
角色：你是一個 SEO 和內容行銷專家

任務：幫我為文章/頁面撰寫優化的標題標籤

文章/頁面主題：[主題]
目標關鍵字：[關鍵字]
目標讀者：[誰]

需求：
1. 提供 5 個不同風格的標題選項
2. 每個標題：
   - 包含目標關鍵字
   - 控制在 55 字以內
   - 有吸引力但不標題黨
3. 標明每個標題的優點

格式：表格呈現
```

## AI 幫你撰寫 Meta Description

Meta Description 是搜尋結果中標題下方的描述文字：

### 撰寫原則

| 要素 | 說明 |
|------|------|
| 長度 | 150-160 字以內 |
| 包含關鍵字 | 自然而然地放入 |
| 總結內容 | 讓人知道會得到什麼 |
| 行動呼籲 | 鼓勵點擊（可選） |

### AI 協助撰寫

```
角色：你是一個專業的 SEO 文案專家

任務：幫我撰寫 Meta Description

頁面標題：[標題]
頁面內容：[簡述內容]
目標關鍵字：[關鍵字]

需求：
1. 提供 3 個版本的 Meta Description
2. 每個版本：
   - 150 字以內
   - 包含目標關鍵字
   - 有吸引力，激發點擊興趣
   - 總結頁面核心價值
3. 標明每個版本的特點
```

## AI 幫你優化內容

### 內容優化檢查清單

```
角色：你是一個 SEO 內容編輯

任務：幫我檢查並優化文章內容

文章標題：[標題]
目標關鍵字：[關鍵字]
文章內容：[貼上文章]

需求：
1. 檢查清單（是/否）：
   - 標題包含關鍵字
   - 第一段包含關鍵字
   - H2/H3 標題包含關鍵字
   - 關鍵字密度適中（1-2%）
   - 有內部連結
   - 有外部連結（可選）
   - 圖片有 Alt 文字
   - 結論有行動呼籲

2. 建議優化：
   - 需要補充的內容
   - 可以刪除的冗詞
   - 標題優化建議
```

### 關鍵字密度優化

```
任務：幫我檢查關鍵字密度並提供建議

文章內容：[貼上文章]
目標關鍵字：[關鍵字]

需求：
1. 計算目前關鍵字密度
2. 建議：
   - 需要增加關鍵字的位置
   - 避免過度堆砌關鍵字
   - 同義詞/相關詞的使用建議
```

## AI 幫你生成結構化資料

結構化資料（Schema Markup）幫助 Google 更好地理解你的內容。

### 常見的結構化資料類型

| 類型 | 用途 |
|------|------|
| Article | 新聞、部落格文章 |
| Product | 產品資訊 |
| LocalBusiness | 當地商家 |
| FAQ | 常見問題 |
| Review | 評論/評分 |

### AI 幫你生成 Schema

```
角色：你是一個精通技術 SEO 的專家

任務：幫我生成 JSON-LD 結構化資料

頁面類型：[文章/產品/FAQ 等]
相關資訊：[填入詳細資訊]

需求：
1. 生成正確格式的 JSON-LD 程式碼
2. 包含所有必要的プロパティ
3. 提供安裝說明（在網站哪裡加入）
4. 用 Google 的 Rich Results Test 驗證的方法
```

## AI 幫你做競爭對手分析

```
角色：你是一個專業的 SEO 競爭分析師

任務：幫我分析競爭對手的 SEO 策略

我的網站：
- 主題：[主題]
- 網址：[URL]
- 主要關鍵字：[關鍵字]

競爭對手：
- 網站 1：[URL]
- 網站 2：[URL]

需求：
1. 分析競爭對手的：
   - 主要關鍵字
   - 內容策略
   - 反向連結概況（如果可以從 Ahrefs/SEMrush 看到）
2. 我可以如何超越他們
3. 我應該優先優化的關鍵字
4. 建議的內容差距（他們有但我沒有的內容）
```

## AI 幫你規劃內容策略

```
角色：你是一個 SEO 內容策略專家

任務：幫我規劃未來 3 個月的內容策略

網站主題：[主題]
目標關鍵字：[主要關鍵字]
目標受眾：[誰]

資源限制：
- 每週可以產出：[數量] 篇文章
- 有人力寫 AI 輔助內容：是/否
- 有預算買圖：是/否

需求：
1. 每月內容主題建議（配合時事/節日）
2. 每篇建議的目標關鍵字
3. 內容類型（教學/列表/比較/案例等）
4. 預估每篇的 SEO 價值（流量潛力）
5. 優先順序建議

格式：用表格和月曆呈現
```

## 圖片 SEO 優化

### 圖片優化要點

| 項目 | 說明 |
|------|------|
| 檔案大小 | <200KB 為佳 |
| 檔案名稱 | 描述性名稱（如 red-apple.jpg） |
| Alt 文字 | 描述圖片內容 + 包含關鍵字 |
| 尺寸 | 預設寬度不要超過 1200px |
| 格式 | WebP 或壓縮後的 JPEG/PNG |

### AI 幫你撰寫 Alt 文字

```
角色：你是一個專業的網頁內容專家

任務：幫我為圖片撰寫 Alt 文字

圖片內容：[描述圖片]
使用頁面：[頁面標題/主題]
目標關鍵字：[關鍵字]

需求：
1. 提供 3 個版本的 Alt 文字選項
2. 每個版本：
   - 簡潔描述（125 字以內）
   - 自然融入關鍵字
   - 符合 SEO 最佳實踐
3. 說明每個版本的適用情境
```

## AI 輔助 SEO 的限制

### AI 能做的

- 關鍵字研究和建議
- 標題、Meta Description 撰寫
- 內容結構建議
- 結構化資料生成
- 基本語法檢查
- 提供優化方向

### AI 不能做的

| 限制 | 說明 |
|------|------|
| 排名保證 | AI 無法影響 Google 的排名演算法 |
| 準確數據 | 搜尋量、競爭程度只是估算 |
| 原創研究 | 無法提供獨特的數據和觀點 |
| 技術實作 | 只能建議，需要你自己設定 |
| 連結建設 | 無法真正建立反向連結 |

## SEO 檢查清單

完成一個頁面或文章後的檢查：

### 技術檢查
- [ ] 頁面載入速度 < 3 秒
- [ ] 手機版顯示正常
- [ ] 有 SSL 憑證 (https://)
- [ ] 沒有重複的 meta 標題和描述
- [ ] 網址結構清楚 (URL Slug 有關鍵字)

### 內容檢查
- [ ] Title Tag 有關鍵字，55 字以內
- [ ] Meta Description 有關鍵字，160 字以內
- [ ] H1 只有一個且有關鍵字
- [ ] 關鍵字出現在文章開頭
- [ ] 關鍵字自然分散在全文中
- [ ] 有使用 H2/H3 標題分段
- [ ] 所有圖片有 Alt 文字
- [ ] 內容長度足夠（有深度）

### 連結檢查
- [ ] 有內部連結到相關頁面
- [ ] 外部連結到權威網站
- [ ] 沒有壞掉的連結
