# 16. 網域註冊與主機選擇

## 什麼是網域和主機

### 網域（Domain）

網域是你網站的「地址」，例如：`example.com`

```
https://www.example.com
      ↑
   這就是網域
```

### 主機（Hosting）

主機是存放你網站檔案的「空間」，就像網站的房子。

```
網站運作的原理：

使用者 → 輸入網域 → DNS 翻譯 → 找到主機 → 顯示網站
              ↑
           網域系統
```

## 網域註冊

### 選擇網域的原則

| 原則 | 說明 | 範例 |
|------|------|------|
| 簡短好記 | 越短越好 | google.com vs gooooogle.com |
| 容易拼寫 | 不要用難拼的字 | flow vs phlow |
| 品牌導向 | 如果是品牌就用品牌名 | starlight.com |
| 避免數字和連字符 | 容易搞混 | my-site.com vs mysite.com |

### 網域購買步驟（以 Namecheap 為例）

```
1. 前往 namecheap.com
2. 在搜尋框輸入想要的網域
3. 檢查是否可用
4. 加入購物車

6. 建立帳號

```

### 購買網域的注意事項

1. **隱私保護**：建議加購 WHOIS 隱私保護，避免個資被公開
2. **自動續約**：開啟自動續約，避免過期
3. **開啟兩步驟驗證**：保護帳號安全
4. **多家比較**：不同後綴、不同供應商價格差很多

## 整合方案：網站平台

如果使用 Wix、WordPress.com、Squarespace 等平台，你不需要另外購買主機，費用已經包含在內。

### 各方案的優缺點

| 方案 | 優點 | 缺點 |
|------|------|------|
| AI 建構平台（Wix 等） | 簡單、快速、全包 | 自訂受限、平台決定你去留 |
| 自架 WordPress | 完全控制、彈性大 | 需要管理主機和更新 |
| 靜態網站 + 托管 | 快速、便宜、安全 | 需要技術能力 |

## DNS 設定基礎

### 什麼是 DNS

DNS（Domain Name System）就像網路的電話簿，把網域名稱翻譯成 IP 位址。

### 常見 DNS 設定

| 記錄類型 | 用途 | 範例 |
|---------|------|------|
| A 記錄 | 指向伺服器 IP | 123.456.78.90 |
| CNAME | 指向另一個網域 | www → @ |
| MX 記錄 | email 伺服器 | mail.example.com |
| TXT | 驗證、SPF 等 | "v=spf1..." |

### 將網域連接到你的網站

**方法 1：使用平台提供的 DNS**

如果是 Wix、Squarespace 等平台：
1. 在平台後台取得「網域設定指示」
2. 登入你的網域管理介面
3. 按照指示新增記錄

**方法 2：使用 Cloudflare**

1. 註冊 Cloudflare
2. 將網域新增到 Cloudflare
3. Cloudflare 會給你 Nameservers
4. 去網域註冊商更新 Nameservers
5. 在 Cloudflare 設定 DNS 指向你的主機

## 實際案例：為新網站購買網域和主機

### 情境

小明想要建立一個個人部落格，網域名稱是 `xiaoming.blog`

### 步驟 1：購買網域

```
1. 前往 Namecheap
2. 搜尋 xiaoming.blog
3. 假設可用，選擇 .blog 或 .com
4. 加入購物車
5. 結帳（假設選擇 .com，約 $12/年）
```

### 步驟 2：選擇主機

```
選擇：SiteGround WordPress 主機
方案：StartUp（$4/月）
續約：$10/月

原因：
- WordPress 官方推薦
- 速度快
- 免費 SSL
- 自動 WordPress 安裝
```

### 步驟 3：設定 DNS

```
1. 在 SiteGround 取得網站 IP 位址（假設 192.168.1.1）
2. 在 Namecheap DNS 設定：
   - A 記錄：@ → 192.168.1.1
   - CNAME：www → @

3. 等待 DNS 生效（通常 24-48 小時）
```

### 步驟 4：安裝 WordPress

```
1. 透過 SiteGround 的一鍵安裝安裝 WordPress
2. 設定管理者帳號密碼
3. 登入 WordPress 後台開始建立網站
```

|------|------|----------|
| 網域 | $12 | $15 |
| 主機 | $48 | $120 |

## 常見問題

### Q：可以換網域嗎？

A：可以，但需要：
1. 購買新網域
2. 設定新 DNS
3. 將新網域連接到主機
4. 設定 301 轉址從舊網域到新網域
5. 更新所有內外部連結

### Q：可以不買網域嗎？

A：可以，但只能用平台提供的子網域（如 `yoursite.wix.com`）。不推薦給正式網站使用。

### Q：什麼時候需要升級主機？

A：當你遇到以下情況時：
- 網站載入速度變慢
- 主機偶爾當機
- 流量接近主機限制
- 需要更多功能（如 SSH、SSH、 cron job）
