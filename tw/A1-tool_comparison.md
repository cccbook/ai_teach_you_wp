## AI 工具選擇決策樹

```
你是完全新手嗎？
├── 是 → Wix ADI 或 Durable
│    └→ Canva + ChatGPT（輔助）
│
├── 有一點技術背景？
│   ├── 想學習程式碼？
│   │   └── Cursor + Claude
│   │
│   └── 只想快速建站？
│       └── Durable 或 Wix
│
└── 是設計師或開發者？
    ├── 想保持設計控制？
    │   └── Framer 或 Webflow
    │
    └── 想自己寫程式碼？
        └── VS Code + GitHub Copilot + Claude
```
