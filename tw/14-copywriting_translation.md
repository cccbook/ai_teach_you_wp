# 14. 讓 AI 幫你寫文案與翻譯

## AI 文案的優勢與限制

### AI 寫文案的優勢

| 優勢 | 說明 |
|------|------|
| 速度快 | 幾秒鐘就能產出初稿 |
| 數量多 | 可以大量快速生成 |
| 變化多 | 可以輕易調整風格和語氣 |
| 不知道寫什麼時 | 可以幫你發想 |
| 不知道怎麼寫時 | 可以給你範例和方向 |

### AI 寫文案的限制

| 限制 | 說明 |
|------|------|
| 缺乏獨特觀點 | 說的是大家都知道的事 |
| 可能不夠精準 | 事實數據需要查證 |
| 語法可能錯誤 | 中文有時會有奇怪的用法 |
| 太過模板化 | 少了人情味 |
| 需要人類把關 | 最終還是要你確認 |

## 正確使用 AI 文案的方式

### 正確姿勢

```
1. AI 幫你發想 → 你選擇方向
2. AI 幫你寫初稿 → 你修改潤飾
3. AI 幫你翻譯 → 你校對調整
4. AI 幫你複製 → 你加入個人特色
```

### 錯誤姿勢

```
✗ 直接複製貼上拿去用
✗ 完全依賴 AI 不做任何修改
✗ 不檢查事實就發布
✗ 期待 AI 完全符合你的風格
```

## AI 文案生成 Prompt 技巧

### 基本 Prompt 結構

```
角色：你是一個專業的 [領域] 文案專家

任務：[具體要寫什麼]

背景資訊：
- [相關背景]
- [目標受眾]
- [產品/服務簡介]

寫作要求：
- 語氣：[正式/輕鬆/專業/親切等]
- 長度：[具體字數]
- 風格：[具體描述]
- 格式：[特殊要求]

禁止：
- [不希望出現的元素]
- 避免：[要避免的風格]

額外要求：
- 加入：[特別要包含的元素]
```

## 常見文案類型 Prompt

### 社群媒體貼文

```
角色：你是一個社群媒體行銷專家，精通
      Instagram、Facebook、Line 的內容策略。

任務：幫我撰寫一週的社群媒體貼文

品牌/個人：[名稱]
領域：[領域]
風格：[風格描述]
本週主題：[本週想傳達的核心訊息]

需求：
1. 每天一個貼文，共 7 篇
2. 每篇包含：
   - 主文案（30-100 字，視平台調整）
   - Hashtag（5-10 個）
   - 配圖建議（optional）
3. 風格多樣，不要全部一樣
4. 互動元素：至少 3 篇要有問問題或 CTA

格式：用表格呈現，清楚標示日期和平台
```

### Email 行銷文案

```
角色：你是一個轉化率優化的 Email 行銷專家。

任務：幫我撰寫一封推廣 Email

背景：
- 發信目的：[新產品/促銷/活動等]
- 目標讀者：[描述讀者]
- 主要訊息：[核心賣點]
- CTA 目標：[讓讀者做什麼]

需求：
1. 主題行（ Subject Line）：提供 5 個版本
2. 預覽文字：配合主題行
3. Email 正文結構：
   - 開頭鉤子（引人繼續讀）
   - 主要內容（1-2 段）
   - 社會證明（可選）
   - CTA 按鈕
4. 長度控制在 200-300 字
5. A/B 測試版本（如果有的話）

格式：模擬實際 Email 的呈現方式
```

### 產品文案

```
角色：你是一個頂尖的電子商務文案專家。

任務：幫我撰寫產品頁面的文案

產品名稱：[名稱]
產品類型：[類型]
目標客群：[描述]
主要功能：[功能列表]
獨特賣點：[USP]

需求：
1. 產品標題（吸睛、包含關鍵字）
2. 產品副標題（一句話說明最大賣點）
3. 產品描述（100-150 字）
   - 開頭：鉤住注意力
   - 中間：功能說明
   - 結尾：為什麼選這個
4. Bullet Points（5-7 個特色點）
5. 常見 FAQ（3-4 個問題）

語氣：專業但不死板，強調好處而非功能
```

### Landing Page 完整文案

```
角色：你是一個高轉換率的 Landing Page 文案專家。

任務：幫我撰寫完整的 Landing Page 文案

產品/服務：[名稱]
目標客群：[描述]
主要價值：[核心賣點]
目標轉換：[讓訪客做什麼]

需求：
1. Hero 區
   - 主標題：痛點/利益切入
   - 副標題：詳細說明
   - CTA 按鈕文字

2. 價值主張區（3-4 個特色）
   - 每個特色：標題 + 描述

3. 社會證明（2-3 個見證）

4. FAQ 區（4-5 個問題）

5. 最後 CTA

長度：完整但精簡，每區塊有實質內容
風格：說服力強但不噁心
```

### 廣告文案

```
角色：你是一個創意十足的廣告文案專家。

任務：幫我撰寫 [Google/FB/IG] 廣告文案

廣告目的：[品牌曝光/產品銷售/報名等]
目標受眾：[描述]
產品/服務：[名稱]
主要訊息：[要傳達的重點]
優惠（如果有的話）：[優惠內容]

需求：
1. 每種格式提供 3 個版本：
   - 標題（30 字以內）
   - 描述（90 字以內）
   - CTA（最多 5 個字）
2. 不同風格：理性/感性/幽默
3. 包含數字、疑問句、對比等技巧

格式：清楚標示每個版本
```

## AI 翻譯技巧

### 翻譯的基本原則

```
英文 → 中文：
- 不要逐字翻譯
- 順暢、自然的中文表達
- 適當調整句構
- 文化轉化（例子、比喻）

中文 → 英文：
- 明確的主詞
- 避免冗詞
- 符合英文習慣用法
```

### AI 翻譯 Prompt

```
角色：你是一個專業的 [中英/英中] 翻譯專家，
      同時精通兩種語言的文化背景。

任務：翻譯以下內容

來源語言：[語言]
目標語言：[語言]
內容類型：[網站/行銷/技術/一般]

原文：
[貼上要翻譯的內容]

翻譯要求：
1. 語意準確，不要遺漏重要資訊
2. 符合目標語言的表達習慣
3. 調整句構讓閱讀順暢
4. 保留專業術語（如果適當）
5. 如果有在地化需求，進行調整

格式：
- 翻譯結果
- 備註（重要術語、翻譯選擇等）
```

### 翻譯後潤飾

翻譯後，請 AI 潤飾：

```
任務：幫我潤飾以下中英文翻譯，
      讓它更自然、更符合母語者習慣。

[貼上翻譯內容]

要求：
1. 檢查文法是否正確
2. 檢查用詞是否自然
3. 檢查語氣是否一致
4. 標註有修改的地方並說明原因
```

## AI 文案的後製處理

### 人工審核清單

拿到 AI 生成的文案後，一定要檢查：

- [ ] 事實和數據是否正確
- [ ] 名稱、術語是否準確
- [ ] 語法錯誤
- [ ] 標點符號
- [ ] 是否有錯別字
- [ ] 邏輯是否通順
- [ ] 風格是否符合品牌調性
- [ ] 是否有敏感內容

### 加入人類元素

AI 文案的問題是「太 AI」了。你可以這樣加入人類元素：

1. **加入你自己的故事**
   ```
   AI：「這款產品採用最新技術，大幅提升效率。」
   人類版：「我試用這款產品的第一週，就省下了 3 小時的時間。」
   ```

2. **加入具體細節**
   ```
   AI：「很多人都有這個問題。」
   人類版：「你是不是也曾經...」（讀者畫面出現）

3. **改變句構和用詞**
   ```
   AI：「我們的產品是最好的選擇。」
   人類版：「用了就回不去了。」
   ```

4. **加入情緒**
   ```
   AI：「感謝您的購買。」
   人類版：「太開心你選擇了我們！」
   ```

## 避免 AI 文案的陷阱

### 陷阱 1：重複率過高

Google 可能會降低重複內容的排名。

**解決方法**：
- 不要完全複製別人的內容
- AI 生成後要修改
- 加入獨特的觀點和經驗

### 陷阱 2：語法奇怪

AI 翻譯或生成的中文有時不自然。

**解決方法**：
- 大聲朗讀一遍
- 請母語者幫忙看
- 用 Google 文件的文法檢查

### 陷阱 3：資訊過時

AI 的訓練資料有截止日期，可能有過時資訊。

**解決方法**：
- 查證重要事實
- 更新最新數據
- 加入 current 事件

### 陷阱 4：版權問題

複製別人的文案再翻譯，可能有侵權問題。

**解決方法**：
- 只參考架構和風格
- 不要照抄
- 加入原創內容

## 提升 AI 文案能力的練習

### 練習 1：風格轉換

```
拿一段 AI 生成的文案，嘗試用不同的語氣重寫：
- 正式 → 輕鬆
- 商務 → 俏皮
- 感性 → 理性

比較哪個版本更適合你的目標。
```

### 練習 2：主題變化

```
用同一個 Prompt 生成 3 個版本，
比較差異，選擇最好的元素组合。
```

### 練習 3：累積反饋

```
每次使用 AI 文案後記錄：
- 哪個版本效果最好？
- 讀者/客戶有什麼反饋？
- 下次可以怎麼改進？

逐漸建立自己的「有效文案庫」。
```

```
